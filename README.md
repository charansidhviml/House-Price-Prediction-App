# 🏠 House Price Prediction

## 📌 Project Overview

This project predicts house sale prices using Machine Learning. It uses the Ames Housing dataset and compares multiple regression models to estimate the selling price of a house based on its features.

---

## 🚀 Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Joblib
- Streamlit

---

## 🤖 Machine Learning Models

- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor

**Best Model:** Random Forest Regressor

---

## 📊 Techniques Used

- Data Cleaning
- Missing Value Handling
- Feature Encoding
- Feature Scaling
- Train-Test Split
- Hyperparameter Tuning (GridSearchCV)
- Model Evaluation
- Model Serialization (Joblib)

---

## 📈 Model Performance

- **MAE:** 18,436
- **R² Score:** 0.874
